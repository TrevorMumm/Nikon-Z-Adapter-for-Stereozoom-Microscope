# Nikon-Z-Adapter-for-Stereozoom-Microscope

Find on printables - https://www.printables.com/model/1539230-nikon-z-adapter-for-microscope

Nikon Z mount to 27.7mm Bausch & Lomb Stereo zoom microscope. This is a pretty niche setup, so not sure how many people will find this useful.

The Z mount design is taken from https://github.com/joe-warren/macro-lens/blob/main/Nikon%20Z-Mount.STEP

The adapter connects to a Nikon Z mount on a Z series camera and then to one of the eyepieces on the microscope. Manual focus on the microscope will control image focus on the camera.

I've included a .3mf file if you are using a Bambu Labs printer, otherwise follow instructions for printing below. The .3mf file contains all three lengths.

I have created 3 different lengths that will give you varying image sizes projected onto the camera sensor. Pictures show the 2xLength model.

Notes:

    Print in Black PETG (DO NOT PRINT IN PLA, or risk damage to your camera if it fails). Black color is not needed but do not use any bright colors, it will affect light and therefore your photo. Black is recommended.

    A tripod to support the weight of the camera is recommended when attaching the setup to the microscope.

Printing parameters:

    Use 4 walls, 35% infill

    Use supports

        I used tree organic supports

    The orientation needs to be with the Z mount facing down (files should be in correct orientation). If printed in the wrong orientation, you may have issues getting it to interface with your camera.
